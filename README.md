# TCGA Expression Rebuild Receipts

Canonical GDC STAR Counts rebuild for 13 TCGA projects.

## Projects
TCGA-ACC, TCGA-CESC, TCGA-CHOL, TCGA-DLBC, TCGA-ESCA, TCGA-KICH, TCGA-MESO, TCGA-PAAD, TCGA-PCPG, TCGA-TGCT, TCGA-THYM, TCGA-UCS, TCGA-UVM

## Schema
gene_id, case_submitter_id, cancer_type_id, raw_count

## Target
60,660 gene rows per sample, GENCODE v36, N_* excluded
